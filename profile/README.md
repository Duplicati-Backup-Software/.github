# Duplicati Backup Software – Encrypted Backup to Disk NAS and Cloud

A free, encryption-first **Duplicati** solution designed to back up files to almost anywhere you already store data. This **Duplicati** software provides scheduled jobs, deduplication-style efficiency, and strong local encryption, allowing users to protect data without surrendering plaintext to storage providers.

## About Duplicati
**Duplicati** is an open-source backup client built for encrypted, versioned archives across diverse backends.  
It provides a browser UI, filters, retention policies, and support for local folders plus popular cloud targets.  
**Duplicati** offers serious backup discipline while remaining free, making it essential for anyone who wants private **Duplicati** offsite copies.
